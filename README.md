# Hytale Mod: Crystal Sword (Create An Item)

Companion mod for the [Hytale Modding Manual](https://hytale-moddings.github.io/hytale-modding-docs/) tutorial on **Create An Item**.

**Repository:** [hytale-moddings/hytale-mods-custom-weapon](https://github.com/hytale-moddings/hytale-mods-custom-weapon)

## What This Mod Does

Adds the **Crystal Glow Sword** — a custom melee weapon with Blockbench model, custom icon, and item definition. This is the base v1.0.0 sword. The advanced version with projectile attacks lives in [hytale-mods-custom-projectile](https://github.com/hytale-moddings/hytale-mods-custom-projectile).

## Installation

1. Copy the mod folder to `%APPDATA%/Hytale/UserData/Mods/CreateACustomWeapon/`
2. The folder name **must** be `CreateACustomWeapon`
3. Start a Hytale server with the mod enabled

## Testing In-Game

```
/spawnitem Crystal_Sword
```
